== Description ==

A plugin to remove all pending comments from your DB in one click. A useful tool if you've been heavily spammed or have not managed your blog for a while and would just like a fresh start for your comments. Note that this will not modify any previous that are not marked as unapproved.

== Installation ==

Standard installtion required.


1. Upload 'Bulk_Comment_Removal.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find the plugin in the 'Tools' menu
